# Historias de Usuario — BC Devices

Esta sección presenta las Épicas e Historias de Usuario identificadas para el bounded context de **Devices (Dispositivos)** de la plataforma Clair IoT. Cada historia está redactada desde la perspectiva de un rol real de la plataforma e incluye criterios de aceptación detallados que describen el flujo de negocio y comportamiento lógico del sistema sin hacer referencia a elementos específicos de la interfaz de usuario.

---

## Epics

| Epic / Story ID | Título | Descripción | Criterios de Aceptación | Relacionado con (Epic ID) |
|---|---|---|---|---|
| WA-EP-02 | Gestión de Organizaciones, Espacios, Dispositivos, Controles y Umbrales | Como Usuario, quiero crear y administrar mis organizaciones y espacios, registrar dispositivos físicos asignándolos a mis zonas, monitorear su estado y telemetría en tiempo real, enviarles comandos de control remoto (standby, wake, restart), y configurar límites de umbrales para las variables del aire, para administrar y supervisar de forma integral mi red de sensores ambientales. | La Épica se completa cuando todas las historias de usuario relacionadas han sido implementadas, validadas y aceptadas. | - |

| WA-US-15           | Eliminar Espacio                                             | Como Usuario, quiero eliminar un espacio físico de mi organización, para quitar las ubicaciones que ya no utilizo. | Dado que el Usuario solicita eliminar un espacio que no tiene dispositivos asignados, cuando confirma la operación, entonces el sistema borra permanentemente el espacio. Dado que proporciona un identificador malformado o incorrecto, cuando solicita el emparejamiento, entonces el sistema rechaza la operación e indica la falla del formato. | WA-EP-02                   |
| WA-US-19           | Renombrar Dispositivo                                        | Como Usuario, quiero cambiar el nombre de un dispositivo registrado, para poder identificarlo fácilmente por su ubicación o función. | Dado que el Usuario es propietario del dispositivo, cuando proporciona un nuevo nombre válido para el mismo y solicita su modificación, entonces el sistema actualiza el nombre del dispositivo en la plataforma. Dado que proporciona un nombre vacío, cuando solicita el cambio, entonces el sistema rechaza la actualización. | WA-EP-02                   |
| WA-US-20           | Eliminar Dispositivo                                         | Como Usuario, quiero dar de baja un dispositivo registrado en la plataforma, para quitar del sistema los sensores que ya no utilizo. | Dado que el Usuario selecciona un dispositivo registrado de su propiedad, cuando solicita y confirma su eliminación, entonces el sistema retira permanentemente el dispositivo y elimina su asociación con cualquier espacio físico de la plataforma. | WA-EP-02                   |
| WA-US-21           | Restablecer Asignación de Espacio del Dispositivo            | Como Usuario, quiero desvincular un dispositivo de su espacio actual sin eliminarlo de la plataforma, para poder reasignarlo a otro espacio en el futuro. | Dado que el Usuario tiene un dispositivo asignado a un espacio, cuando solicita restablecer su asignación, entonces el sistema remueve la asociación entre el dispositivo y el espacio físico, dejando al dispositivo en estado "Sin asignar" pero conservando su registro en la plataforma. | WA-EP-02                   |
| WA-US-22           | Ver Dispositivos por Espacio                                 | Como Usuario, quiero obtener el listado de dispositivos registrados en un espacio físico determinado, para conocer qué sensores están asignados a dicha área. | Dado que el Usuario selecciona un espacio físico, cuando solicita consultar sus dispositivos, entonces el sistema devuelve el listado de todos los dispositivos asignados a ese espacio, especificando su nombre, número de serie, estado, tipo de dispositivo, identificador de hardware y fecha de última conexión. Dado que el espacio no tiene dispositivos, cuando se realiza la consulta, entonces el sistema devuelve un resultado indicando la ausencia de dispositivos. | WA-EP-02                   |
| WA-US-23           | Ver Detalles del Dispositivo                                 | Como Usuario, quiero consultar la configuración y los parámetros del sistema de un dispositivo registrado, para evaluar sus detalles técnicos. | Dado que el Usuario solicita los detalles de un dispositivo mediante su identificador, cuando el sistema procesa la consulta, entonces devuelve el número de serie, nombre, estado, espacio asignado, tipo de dispositivo, identificador de hardware, propietario, fecha de activación y última fecha de actividad. Dado que el dispositivo no existe en la plataforma, cuando se realiza la consulta, entonces el sistema notifica que el dispositivo no fue encontrado. | WA-EP-02                   |
